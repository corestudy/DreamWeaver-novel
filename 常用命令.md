# 常用命令（Windows）

## 开发与检查
- 开发启动：`npm.cmd run dev`
- 桌面运行：`npm.cmd run desktop`
- 语法检查（含接口文档一致性校验）：`npm.cmd run check`
- 仅接口文档一致性检查：`npm.cmd run check:api-doc`
- 原生模块重建（Node ABI）：`npm.cmd run native:rebuild:node`
- 原生模块重建（Electron ABI）：`npm.cmd run native:rebuild:electron`
- 原生模块自检：`npm.cmd run native:check`

## 清理（轻量化）
- 清理本地配置/数据：`npm.cmd run clean:user-data`
- 清理打包产物：`npm.cmd run clean:dist`
- 一键轻量化：`npm.cmd run lightweight`

## 打包 EXE
- 常规打包：`npm.cmd run build:desktop`
- 一键测试后打包（推荐）：`npm.cmd run build:desktop:ready`
- 推荐打包（跳过签名编辑，规避权限问题）：

```powershell
$env:CSC_IDENTITY_AUTO_DISCOVERY='false'; npm.cmd run build:desktop -- --config.win.signAndEditExecutable=false
```
